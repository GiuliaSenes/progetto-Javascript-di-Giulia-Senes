# 🔢 Simple Dynamic Counter
Un'applicazione web minimalista creata in **JavaScript puro** (Vanilla JS) che permette di incrementare, decrementare e resettare un contatore. L'interfaccia è generata interamente in modo dinamico tramite il DOM.

## ✨ Funzionalità
- **Incremento/Decremento**: Cambia il valore in tempo reale.
- **Reset**: Riporta il valore a 0 con un click.
- **Limite Zero**: Il contatore non scende sotto lo zero.
- **Responsive Design**: Ottimizzato per mobile e desktop grazie a Bootstrap 5.

## 🛠️ Tecnologie Utilizzate
- **JavaScript**: Logica e manipolazione dinamica del DOM.
- **HTML5/CSS3**: Struttura e personalizzazione di base.
- **Bootstrap 5**: Layout responsive e componenti UI.
